박스모델

박스모델이란 웹 문서내용을 박스 형태로 표현하는 것을 말한다

나열형태에 따라 블록과 인라인이 있다

블록 : 태그 하나가 너비 100%로 한 줄을 모두 차지하여 블록태그 옆에는 다른 요소가 위치할 수 없다

인라인 : 너비가 태그의 컨텐츠 만큼에 해당하여
한 줄에 인라인 요소 여러개가 위치하게 할 수 있다

블록(block) div, p, form, hr, table, ul, ol, h1~h6
인라인(inline) span, strong, label, a : inline
input, textarea, button, img : inline-block

박스모델 블록의 속성

  • content : 텍스트나 이미지
  • padding : 안쪽 여백으로 border와 content 사이의 영역
  • border : 테두리
  • margin : 바깥 쪽 여백으로 태그와 태그 사이의 영역
