시맨틱 태그

div태그와 다른 것이 없는데 이름이 달라 쉽게 알 수 있다는 장점

태그의 이름만으로도 역할을 짐작할 수 있도록 내부구조 파악이 용이한 div인 태그

header 태그, footer 태그, nav 태그(네비게이션), section 태그(content), aside 태그(side영역),
article 태그(콘텐츠가 여러개 일때 section 내부에 사용하는 각각의 컨텐츠를 의미하는 태그),
address 태그 : footer 내부에 주소를 표현하는 태그
nav 태그(네비게이션) : header 태그 내부에 메뉴를 표현하는 태그
