속성선택자

선택자[속성] : 속성을 갖는 모든 요소를 선택

선택자[속성=데이터값] : 속성값이 데이터값인 요소를 선택

선택자[속성!=데이터값] : 속성값이 데이터값이 아닌 요소를 선택

선택자[속성^=데이터값] : 속성값이 데이터값으로 시작하는 요소를 선택

선택자[속성$=데이터값] : 속성값이 데이터값으로 끝나는 요소를 선택

선택자[속성*=데이터값] : 속성값에 데이터값이 포함된 요소를 선택

선택자[속성~=데이터값] : 속성값에 데이터값이 단어로 포함된 요소를 선택
