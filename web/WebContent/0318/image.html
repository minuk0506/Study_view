링크만들기

a태그 : 링크는 현재 웹 페이지에서 다른 페이지로 연결하기 위한 형태
a태그로 링크를 적용한다
target : 어떻게 열 것 인지
target 속성값 blank : 새 창으로 열겠다
target 속성값 self : 이 창에서 열기(지정하지 않으면 기본이 셀프)
현재 페이지에서 특정 페이지로 연결할 수도 있는데 이런 형태를 앵커라고 한다

a태그에 onclick="alert(내용)"
(onclick : 클릭시 이벤트)
(alert() : alert창 띄우기)
