table : 행열로 이루어진 표의 형태를 만들때 사용하는 태그
tr: table row - 행에 해당하는 태그
td: table data - 행 내부의 열에 해당하는 셀 태그
th : table head - 행 내부의 제목에 해당하는 셀 태그
