html : HyperText Markup Language 웹브라우저로 표현하고 실행하기 위한 웹 문서를 만드는 언어
<> 부등호기호로 둘러싸인 요소를 태그라 한다
element(요소) :html 문서를 구성하는 약속된 단어
style : 웹 문서를 보기좋게 꾸미는데 사용
css : Cascading Style Sheet (스타일)
